# Discover Skills

## Procedure

1. Read workflow history
2. Always Ask the user how far away from workflow history you can use
2. Look for repeated command sequences.
3. Count occurrences.
4. If a sequence appears 3 or more times:
   - Suggest saving into memor.
5. Present suggestion to user.
